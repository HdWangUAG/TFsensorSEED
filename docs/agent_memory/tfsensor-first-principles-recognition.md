TFsensorSEED Stage-1e (2026-06-14): mined PDBe atom-level interactions across ALL PDB structures binding each panel steroid (STR=progesterone 47 instances, TES=testosterone 44, EST=estradiol 99, HCY=cortisol 17). Tool: `tfsensor/pdb_interaction_mining.py`; results in `results/stage1e_pdbmine/` (interaction_summary.json + 2 pub figures + FIRST_PRINCIPLES_BINDING.md). Steroids recognized by H-bonds to their oxygens on a hydrophobic ring core; the *partner residue type* per oxygen is the specificity signal.

**Recognition code (data-driven):** estradiol **3-OH phenol → GLU 42% + ARG 20%** (phenol DONATES to a carboxylate clamp — the unique estradiol signature; canonical ERα Glu353/Arg394 + His524-on-17OH motif). 4-en-3-one **3-keto → ARG/GLN** (keto only ACCEPTS). Cortisol's 5 oxygens (3-keto,11β-OH,17α-OH,20-keto,21-OH; verified via HCY bond connectivity O1-O5→C3/C11/C17/C20/C21) are 82-100% polar, served by SER/THR/ASN/ARG → demands a polar pocket.

**Closure with WT/mutant biology:** WT pocket already has **ARG123 (2.9Å) + GLU106 (2.66Å), GLN88, SER144, GLN151**. Arg123 reading the 3-keto explains WT testosterone/progesterone response and why estradiol (needs a Glu-clamped phenol, not Arg-keto) is the non-responder. **L147R** adds Arg (recurring cortisol partner) → cortisol gain/preference; **F119W** Trp adds packing+indole-NH+aromatic surface → sensitivity boost. The PDB stats mechanistically rationalize both engineered phenotypes.

**Structural check DONE (Boltz holo, distances ligand-O → sidechain):** the A-ring oxygen sits ~3.5 Å from **R123** for estradiol's 3-OH phenol and ~4.7 Å for testosterone's 3-keto — i.e. **R123 (a donor) is the A-ring reader**. That is favorable for the keto *acceptor* decoys but a **donor–donor mismatch for estradiol's phenol** → first-principles reason WT rejects estradiol. The D-ring 17-O sits ~6 Å from Q88. **E106 is 7.3–15.6 Å from both ligand oxygens — too far to engage** (peripheral, not a real clamp despite the 2.66 Å contact-set label). PyMOL figures: `results/stage1e_pdbmine/pymol/pocket_{estradiol,testosterone}.png` (+ .pse); tool `tfsensor/pymol_pocket.py` (dual labels, exp = model + 9).

**How to apply (estradiol design recipe, for paused Stage 3):** the crux residue is **R123 (exp R132)** — swap the A-ring reader from **Arg→Glu/Asp** so a carboxylate ACCEPTS estradiol's phenol O-H (estrogen-receptor motif), which simultaneously REJECTS the keto decoys (no donor). Add a polar/His near 17β-OH; keep ring-core hydrophobic. Converts design from blind LigandMPNN to a hypothesis-driven pocket graft with a validatable positive control. See [[tfsensor-stage1-triple-nogo]], [[tfsensor-mutant-benchmark]].
