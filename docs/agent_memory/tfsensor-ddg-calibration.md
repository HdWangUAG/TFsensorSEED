TFsensorSEED Stage-3a built a fixed-backbone ΔΔG-of-binding arbiter (`tfsensor/ddg_mutation.py` worker, `ddg_panel.py` parallel orchestrator, `ddg_report.py`, `drive_ddg.sh`; results in `results/stage3_ddg/`). Threads R123E/R123D (model 123 ARG, estradiol phenol-clamp graft), F119W (model 119 PHE), L147R (model 147 LEU) into WT Boltz holo backbones, both homodimer chains; ΔΔG = dG_separated(mut) − dG_separated(WT) with matched per-site WT baseline, ligand anti-ejection tether (weak harmonic CoordinateConstraint sd 2.0 Å, relax-time only).

**Attempt 1 result (2026-06-14): calibration FAILED, 0/10.** Diagnosis: the ligand tether WORKED (0 ejections, COM drift <0.6 Å), but **9/12 WT baseline dG_separated are POSITIVE/clashing (up to +67 REU; cortisol worst)** — the strict-fixed-backbone + single-cycle neighborhood-only FastRelax cannot resolve raw Boltz-pose clashes, so dG is unreliable and ΔΔG = difference of two bad numbers (27% of per-seed |ΔΔG|>5 REU; one −25). Only estradiol gave clean negative baselines. Stage-1c's *full* CA-constrained relax got clean −19..−29 for the same ligands, so the fix is known.

**How to apply (revised protocol — flex-ddG style, still no FEP):** (1) **pre-relax the WT holo complex thoroughly** per backbone → clean negative reference before measuring ΔΔG; (2) **local backbone flexibility (backrub)** in the mutation step so bulk-increasing F→W/L→R are accommodated (strict fixed-backbone is too rigid for them); (3) **ensemble averaging ~10–20 models, report median** to beat per-estimate noise. Then re-test calibration (L147R penalize test/favor cortisol; F119W boost test/prog) before trusting it to rank Stage-3 grafts. See [[tfsensor-mutant-benchmark]], [[tfsensor-first-principles-recognition]].


## v2 (flex-ddG) outcome — 2026-06-14
Rebuilt as flex-ddG: WT pre-relax (clean baselines, now −19..−27 incl. cortisol −20 vs +42..+67 before) + backbone-flexible ensemble (shell incl. site±2; mutant minimizes its OWN local backbone after mutation, fixing small→large false-negatives) + ligand tether + PAIRED WT/mut per member + **median across seeds** (1 R123D seed was a −6.8 outlier; mean misled → use median + sign-agreement). Cost: 12 (lig×seed) workers ×8-member ensemble, all parallel on 36 cores (~1 wave).

**Robust findings (all 3 seeds agree):** (1) R123E & R123D **penalize progesterone** (+1.2/+1.8) — carboxylate↔keto repulsion is real; (2) **L147R penalizes testosterone** (+1.11) and makes **cortisol the most-favored ligand (rank #1)** — reproduces the experimental L147R→cortisol specificity-switch DIRECTION. NOT robust: single R123E/D does NOT rescue estradiol (estradiol ranks #3; the carboxylate instead favors **cortisol** via its polyol) and does not penalize testosterone; **F119W shows no robust ΔΔG** (sensitivity/EC50 effect, out of specificity scope). 

**Verdict:** instrument is mechanically sound + trustworthy as a RELATIVE specificity ranker (got the L147R switch); use as **Tier-1** (10k→Top10). **Key design implication:** a single R123E/D is INSUFFICIENT for estradiol — need the full Glu/Asp+Arg+His clamp (per PDB), AND must guard against cortisol hijacking the carboxylate. Two-tier pipeline documented in `PIPELINE.md` (Tier-2 = Boltz→drMD→FEP on Top-10, not yet built). See [[tfsensor-first-principles-recognition]].

## Two-state (apo→holo) efficacy test — 2026-06-14
Folded matched mutant APO (WT/L147R/F119W) with Boltz-2 (`--apo` mode in boltz_holo_inputs; `drive_apo.sh`; `tfsensor/two_state_panel.py`; results in `results/stage3_apo/`). opening = mean DBD(holo)−mean DBD(matched apo) at 37/40. **Verdict: single-structure DBD opening does NOT reliably predict fluorescence amplitude.** The decisive L147R test FAILED — predicted testosterone(+6.99)>cortisol(+6.62)>progesterone(+6.36) vs experimental cortisol>progesterone>testosterone≈0; the three openings differ <0.65 Å while per-model DBD std is 1.4–4.0 Å → indistinguishable (Spearman −0.2). WT Spearman −0.4 (cortisol opens most but doesn't bind WT → opening must be gated by binding). Only ROBUST signal: estradiol opens least in every variant (weak activator) → usable only as a COARSE dead-binder gate, not a quantitative amplitude ranker. Mechanism: efficacy is a conformational free-energy/population quantity, unresolvable from static prediction (~0.5 Å signal vs several-Å noise); would need enhanced-sampling MD (drMD tier). **Decision: amplitude stays WET-LAB (scope boundary confirmed); Stage-3 computational filter = Tier-1 ΔΔG specificity + Tier-2 FEP; two-state opening kept only as qualitative dead-binder check.** See [[tfsensor-first-principles-recognition]].
